# Digital Forensics Investigation Lab

## Overview

This project presents a digital forensics investigation. The objective was to collect, analyze, and interpret digital evidence using industry-standard forensic tools and methodologies.

## Tools Used

* Autopsy
* FTK Imager
* Volatility Framework
* Epoch Timestamp converter

## Objectives

* Acquire and preserve digital evidence
* Perform disk and memory analysis
* Identify potential malicious activities
* Document findings in a structured report

## Key Activities

* Disk image acquisition and analysis
* File system examination and recovery
* Memory dump analysis

## Key Findings

* Recovered deleted files and artifacts
* Identified suspicious activity
* Detected potential indicators of compromise (IoCs)
* Analyzed system behavior through memory forensics

## Skills Demonstrated

* Digital Evidence Handling
* Disk & Memory Forensics
* Use of Forensic Tools
* Incident Investigation & Reporting

Full report available in the repository

## Environment

* Selected infected virtual machine
* Isolated environment for safe analysis

This project was conducted for educational purposes in a controlled environment.
No real-world systems or personal data were involved.
